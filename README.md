### Tools Used

- Pandas
- NumPy

### Dataset Used
- Adult Income Dataset

### Work Done
- Loaded the Adult Income dataset using Pandas
- Identified categorical and numerical features
- Applied Label Encoding for ordered categorical features
- Applied One-Hot Encoding for non-ordered categorical features
- Scaled numerical features using StandardScaler
- Compared data before and after scaling
- Saved the preprocessed dataset for future modeling

### Key Learnings
- Encoding converts categorical data into numerical form
- Label Encoding is used when categories have order
- One-Hot Encoding is used when no order exists
- Feature scaling helps improve model performance
- Algorithms like KNN, SVM, and Linear Regression need scaling
